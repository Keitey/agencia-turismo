# Site agência de turismo

Este é um projeto de um site fictício de uma agência de viagem criado em <strong>React</strong> e usando <strong>Material UI</strong>.

* Icons-material;
* Styled UI;
* Palette;
* Darkmode;
* Typography;
* Button;
* ButtonGroup;
* Checkbox;
* FAB - Floating Action Button;
* Switch;
* TextField;
* Tranfer list;
* Avatar;
* Badge;
* Material icons;
* List;
* Tooltip;
* App Bar;
* Card;
* Menu;
* Box;
* Stack;
* Image List;
* Modal;
